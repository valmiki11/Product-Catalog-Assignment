▶️ How to Run the Backend
Unzip the project

Open in any Java IDE (IntelliJ, VS Code, Eclipse)

Run ProductCatalogApplication.java

Visit: http://localhost:8080/api/products to view JSON products

(Optional) Access DB at http://localhost:8080/h2-console

JDBC URL: jdbc:h2:mem:productdb

User: sa, Password: (leave blank)
